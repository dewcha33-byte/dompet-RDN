Arti Status Transaksi RDN Offline

Fokus memahami pembukuan instan dalam jaringan asimetrik rupiah digital:

PENDING (MENCOCOKKAN)

Definisi status Pending berarti pemindahan dana digital telah berhasil dilakukan antar chip perangkat pembayar dan penerima secara peer-to-peer tanpa internet. Berhasil secara fisik didebit tetapi menunggu online untuk pelaporan pembukuan di server BI.

SELESAI (TERREKONSILIASI)

Definisi status Selesai berarti log transaksi telah sukses dikirim ke gateway Bank Indonesia dan lunas divalidasi pembukuannya. Saldo resmi dikonversikan ke dalam mutasi umum yang terdaftar resmi.

Teknologi Secure Element RDN

Bagaimana modul perangkat keras mendedikasikan proteksi mutlak:

Hardware Locked

Kunci privat unik disolder khusus pada logic board chip, anti-copy dan tidak bisa dimodifikasi oleh virus software.

Anti-Tampering Sensor

Sistem auto-destruct kunci secara logikal mendeteksi jika ada anomali pembongkaran hardware fisik chip koin.

FIDO2 Biometric Sync

Memindai scan sensor biometrik wajah langsung ke hardware terisolasi untuk otorisasi debet instan.
